HTML 常见基础标签

标签 – tag

Tag <html> <head> <meta>
分为单标签 <meta> 双标签<html> <head> <body> <title>

  1. 标签是HTML语言基本组成部分
  2. 标签是分层的；多标签总是成对出现，包括开始标签和结束标签
  3. 根标签是html，包括两个子标签head 和 body
  4. 标签与属性不区分大小写 如 <html> <HTML> 都可以

属性与属性值 – attribute

  1. 附属于标签，对标签的特征进行设置
  2. 属性的值可以用双引号引起来 （单引号也可以）
  3. 属性和属性值不区分大小写
  4. 属性之间无先后次序，属性也可以省略 -> 默认
  5. 属性都是写在开始标签中
  • hr是单标签 — 一条横线
  • px – 像素
  • 分辨率 1366 * 766 -> 横向有1366个像素点 纵向有766个像素点
  • width -> 长度 size -> 粗细 align -> 位置
  • 忘记属性 www.w3school.com.cn -> 线上帮助文档

文本 – text

  • 标签的内容可以是子标签或者普通文本
  • 最内层标签的内容称为文本
  • 外层标签的内容也可以称为文本
  • 包括空格与换行

元素 – elemen

  • 开始标记+中间内容+结束标记
  • <p align="center"></p> HTML文档包括两部分：头部部分和主体部分
  • 一个完整的标签我们也称之为网页上的一个元素

标题标签和段落标签

标题标签

  • h1-6 一共有六级标题

段落标签

  • <br> 放在哪里就在哪里换行 -> 放在<p></p>内帮助其换行
  • <p></p> paragraph 段落标签 被p包含的文字是一段 内部无换行 每一对独立占一行

格式化标签

https://www.w3school.com.cn/html/html_formatting.asp – 文档地址

  • <b></b> 变粗
  • <big></big> 变大
  • <small></small> 变小
  • <em></em> 着重 -> 倾斜
  • <i></i> 倾斜
  • <strong></strong>
  • <sub></sub> 下标签
  • <sup></sup> 上标签
  • <del></del> 删除标签
  • 复合标签，效果叠加
    • 注意嵌套格式，谁包谁无所谓，但是开始结束后要对应
    • 合理嵌套
