HTML基本结构

  • 根标签 <html></html> 所有其他标签都放在该标签中 （html4 中前面有一串版本约束）
    • html 根标签中有两个一级子标签 分别是head body
    • ctrl+l查找
  • <head></head>头标签 – 网页头上菜单栏中显示的部分,通常不展示在网页上，却有一些重要内容
    • <title></title>
  • <body></body>体标签 – 显示在页面上的内容部分

ps:换行不会被展现在浏览器中
ps:还可能会有js的代码

为什么会有乱码

  • 数据存储在硬盘，看不见
  • 可以看见分区C、D、E盘
  • 硬盘上有许多小磁铁，分为N极与S极，存储数据时，就是改变这些小磁铁的磁极，将其区分为N极与S极。及代表0/1–最终存储为磁盘上的0/1。
  • 必须遵守存储规则–字符集（字典）：代表文字转换对应关系。
  • 字符集不止一个，如GBK(国标，专门处理中文)与UTF-8（既有中文、又有国际通用字符）

怎么处理乱码

  • 弄清存储的字符集是哪个

    • 另存为 -> ANSI（本地默认字符集）

  • 告诉浏览器用的编码 meta – 单标签，有开头无结尾，多一个斜线

  • 处理打开问题 -> 运行配置 -> 编辑 -> 找浏览器启动路径 -> 找浏览器.exe文件位置

注释问题

还可以选中文字按快键ctrl + shift + / ，也可以这样取消注释

head中的标签

  • <title></title> 标题标签 定义显示在网页左上角的标题

  • <meta /> 定义一些当前网页的特殊属性，通过meta标签的属性定义页面的特性（如chrset = ""）

    • <标签名 属性1="" 属性2=""></标签名>

    • charset属性 定义当前页面的字符集 告诉浏览器以什么字符集解析当前页面

      • 早期版本<meta http-equiv="content-type" content="text/html; charset=UTF-8" />

    • name属性 定义当前网页的一些简单描述 便于搜索引擎上根据描述搜索到当前网页

属性 作用
keywords 关键字
author 作者
description 描述一段陈述性的文字，描述当前网页
  • http-equiv refresh 刷新
  • link 学习CSS样式的 引入外部CSS样式文件
  • <script></script> 学习js脚本语言 引入脚本语言文件

ps:单引号、双引号无所谓，但是必须成对出现

body标签

  • bgcolor 背景色
